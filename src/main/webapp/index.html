Why coinmines?
Why use our Mining Pool? We're not saying you should use it, but we're saying that you can. Since we are under heavy development we cannot guarantee yet to be problem free, but we're certainly trying to. Our site is built with Lift, a Scala web framework. This interface is also released under a BSD License on github, so feel free to donate if you like it.
