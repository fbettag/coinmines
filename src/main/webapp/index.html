Why coinmines?
Why use our Mining Pool? This site is under constant monitoring and development, we work hard to update the software packages seamlessly to give you the best pooled mining performance and experience. Our site is built with Lift, a Scala web framework, which makes it extra secure by itself, but we've also put alot of thought into how to keep your data safe. This interface is also released under a BSD License on github, so feel free to donate if you like it.
